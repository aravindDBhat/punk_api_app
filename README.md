# Punk-API-App

website using React.js

# Tech Stack

- React Js
- Bootstrap
- CSS

# Installation

Make sure you have Node.js and npm install.

1.  Clone or Download the repository

```bash
    git clone https://github.com/aravindDBhat/punk-api-app.git
    cd punc-api-app
```

2. Start Frontend

```bash
npm install
npm run start
```

Application runs from localhost:3000.
